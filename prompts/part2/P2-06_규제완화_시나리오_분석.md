| 주담대 한도 (15억↓) | 6억 원 | - | - |

## DSR 계산 공식

```
DSR = (연간 원리금 상환액 ÷ 연소득) × 100%

연간 원리금 상환액 = 대출원금 × (금리 + 스트레스 가산) ÷ 상환기간 계수
```

## 스트레스 DSR 가산금리 영향

| 스트레스 DSR | 실질 심사 금리 | 대출 가능액 영향 |
|-------------|--------------|----------------|
| +3.0%p (현행, 규제지역) | 주담대 + 3.0% | 기준 |
| +1.5%p (완화 시나리오) | 주담대 + 1.5% | +15~20% |
| +0.75%p (대폭 완화 시) | 주담대 + 0.75% | +25~30% |
| 0%p (폐지 시) | 주담대 금리만 | +35~40% |
