# 프롬프트 01: 최신 부동산 정책 요약 (실습 7)

**유형**: Type A - 직접 실행형
**출처**: Part 4 실습 7
**난이도**: 기본
**목표**: 복잡한 정책을 AI로 빠르게 요약하고 핵심 포인트 추출

---

## 프롬프트 구조

### 1. Role (역할)

```
당신은 15년 경력의 부동산 정책 분석 전문가입니다.
복잡한 규제를 쉽게 설명하고 투자자별 시사점을 도출합니다.
```

### 2. Mission (임무)

```
현재 적용 중인 부동산 정책을 종합 정리하고,
투자자 유형별 핵심 시사점을 도출해 주세요.
```

### 3. Data Collection (데이터 수집)

```
다음 정책 데이터를 먼저 조회하세요:

[대출 규제]
- LTV, DTI 규제 수치 (금융위원회, 최신)
- 스트레스 DSR 가산금리 (수도권/비수도권)
- 주담대 한도 제한 (주택가격대별)

[규제지역]
- 투기과열지구 지정 현황 (국토부)
- 조정대상지역 지정 현황 (국토부)
- 토지거래허가구역 현황

[세금]
- 취득세율 (주택 수별, 규제지역 가산)
- 양도세 (1주택 비과세, 다주택 중과)
- 종부세 (공제 기준, 세율)

[정책대출]
- 디딤돌 대출 조건 (소득, 한도, 금리)
- 신생아 특례 대출 조건
```

### 4. Rules (규칙)

```
[최신성 원칙]
- 반드시 최신 정책 기준일 명시 (예: 2025.10.15 대책)
- 유예 기간 있는 규제는 종료일 명시

[정확성 원칙]
- 수치는 공식 출처 기반
- 추측 표현 사용 금지
- 변경 가능성 있는 항목 별도 표기

[실용성 원칙]
- 투자자 유형별 시사점 필수
- 실제 적용 예시 포함
```

## 플레이스홀더 버전

```
당신은 15년 경력의 부동산 정책 분석 전문가입니다.
복잡한 규제를 쉽게 설명하고 투자자별 시사점을 도출합니다.

[현재 연월] 기준 적용 중인 부동산 정책을 종합 정리해 주세요.

## 분석 요청

**1. 대출 규제 현황**
- LTV, DTI, DSR 규제 수치
- 스트레스 DSR 가산금리
- 주담대 한도 제한 (주택가격별)
- 규제지역 vs 비규제지역 차이

**2. 규제지역 현황**
- 투기과열지구 지정 지역
- 조정대상지역 지정 지역
- 토지거래허가구역 현황

**3. 세금 규제**
- 취득세 (주택 수별)
- 양도세 (1주택/다주택)
- 종부세 (공제 기준)

**4. 정책대출 현황**
- 디딤돌 대출 조건
- 신생아 특례 대출 조건

**5. 투자자별 시사점**
- 무주택자, 1주택자, 다주택자 각각

## 출력 형식
각 항목을 표로 정리하고, 핵심 포인트를 3줄로 요약해 주세요.
```

## 완성형 예시

```
당신은 15년 경력의 부동산 정책 분석 전문가입니다.
복잡한 규제를 쉽게 설명하고 투자자별 시사점을 도출합니다.

**2025년 12월** 기준 적용 중인 부동산 정책을 종합 정리해 주세요.

## 분석 요청

**1. 대출 규제 현황 (2025.10.15 대책 기준)**
- LTV 40%, DTI 40% (규제지역 기준)
- 스트레스 DSR: 규제지역 +3.0%p, 비규제지역 +1.5%p, 지방 +0.75%p
- 주담대 한도: 15억 이하 6억, 15~25억 4억, 25억 초과 2억
- 규제지역 vs 비규제지역 대출 한도 차이

**2. 규제지역 현황 (2025.10.16 지정 기준)**
- 투기과열지구: 서울 25개구 전체
- 조정대상지역: 서울 + 경기 12개 지역 (과천, 광명, 성남3구, 수원3구, 안양동안, 용인수지, 의왕, 하남)
- 토지거래허가구역: 투기과열지구 전체

**3. 세금 규제**
- 취득세: 1주택 1~3%, 2주택 8%, 3주택 12% (규제지역)
- 양도세: 다주택 중과 2026.05.09까지 유예
- 종부세: 1주택 12억 공제, 다주택 9억 공제

**4. 정책대출 현황**
- 디딤돌: 소득 6천만원, 한도 2~3.2억, 금리 2.45~3.55%
- 신생아 특례: 소득 1.3억, 한도 4억, 금리 1.8%

**5. 투자자별 시사점**
- 무주택자: 신생아 특례/디딤돌 활용, 생애최초 취득세 감면
- 1주택자: 비과세 요건(2년 거주), 일시적 2주택 처분기한(2년)
- 다주택자: 중과 유예 기간 활용, 규제지역 취득세 주의

## 출력 형식
각 항목을 표로 정리하고, 핵심 포인트를 3줄로 요약해 주세요.
정책 기준일과 출처를 명시해 주세요.
```

## 후속 프롬프트 (Type D)

### D-1: 특정 규제 상세
```
"스트레스 DSR 규제를 더 자세히 설명해 주세요. 실제 대출 한도에 어떤 영향을 미치나요?"
```

### D-2: 규제지역 상세
```
"경기도 규제지역 12곳의 규제 내용을 상세히 비교해 주세요."
```

### D-3: 정책대출 비교
```
"신생아 특례 대출과 디딤돌 대출, 어떤 상황에서 어느 쪽이 유리한지 비교해 주세요."
```

- [ ] LTV/DTI/DSR 수치 정확성 (금융위 기준)
- [ ] 규제지역 목록 최신성 (국토부 기준)
- [ ] 세금 규제 수치 정확성 (기재부 기준)
- [ ] 정책대출 조건 최신성 (주택금융공사 기준)
- [ ] 모든 수치에 기준일 명시
- [ ] 투자자 유형별 시사점 도출
- [ ] 핵심 3줄 요약 포함
