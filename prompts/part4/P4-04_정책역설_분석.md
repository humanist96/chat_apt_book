# 프롬프트 04: 정책 역설과 투자 기회 분석 (실습 9)

**유형**: Type A - 직접 실행형
**출처**: Part 4 실습 9
**난이도**: 중급
**목표**: 정책 역설에서 발생하는 투자 기회 분석

---

## 프롬프트 구조

### 1. Role (역할)

```
당신은 15년 경력의 부동산 정책 분석 전문가입니다.
정책의 의도와 실제 시장 반응의 차이를 분석하고,
역설에서 발생하는 투자 기회를 발굴합니다.
```

### 2. Mission (임무)

```
현재 부동산 정책들의 역설(의도 vs 실제 효과)을 분석하고,
정책 수혜/주의 섹터와 투자자 유형별 전략을 제시해 주세요.
```

### 3. Data Collection (데이터 수집)

```
다음 정책별 시장 반응 데이터를 조회하세요:

[6.27 대책 - 주담대 6억 제한]
- 정책 시행 전후 거래량 변화
- 10~15억 매물 시장 반응
- 풍선효과 발생 지역

[10.15 대책 - 스트레스 DSR 3.0%p]
- 대출 가능액 변화 (실제 사례)
- 매물잠김 현상 분석
- 실수요자 영향

[토지거래허가구역 확대]
- 지정 지역 거래량 변화
- 비규제 인접 지역 반응
- 풍선효과 수혜 지역

[시장 반응 지표]
- 지역별 거래량 변동 (한국부동산원)
- 매물 증감 현황
- 호가 변화 추이
```

### 4. Rules (규칙)

```
[역설 분석 기준]
1. 정책 의도: 공식 발표 목적
2. 실제 반응: 시장에서 나타난 결과
3. 역설 현상: 풍선효과, 매물잠김, 가격 역전 등

[수혜/주의 섹터 판단]
- 수혜 섹터: 규제 우회 가능, 풍선효과 수혜
- 주의 섹터: 규제 직접 적용, 유동성 감소

[투자자 유형 구분]
- 무주택 실수요자
- 1주택 갈아타기
- 다주택 투자자
- 법인 투자자
```

## 플레이스홀더 버전

```
당신은 15년 경력의 부동산 정책 분석 전문가입니다.

- [정책 2] (예: 스트레스 DSR)

**2. 정책 수혜 지역 분석**

- 대출 활용 가능 지역

**3. 정책 주의 섹터 분석**
현재 정책 환경에서 불리한 섹터:
- 초고가 주택

**4. 투자자 유형별 전략**
무주택자, 1주택자, 다주택자, 법인별 유불리와 전략

**5. 정책 변화 모니터링 포인트**

## 완성형 예시

```
당신은 15년 경력의 부동산 정책 분석 전문가입니다.
정책의 의도와 실제 시장 반응의 차이를 분석하고,
역설에서 발생하는 투자 기회를 발굴합니다.

**2025년 12월** 기준 부동산 정책의 역설과 투자 기회를 분석해 주세요.

## 분석 요청

**1. 정책 역설 분석**
다음 정책들의 의도와 실제 시장 반응을 비교해 주세요:
- **6.27 대책** (주담대 6억 제한): 가계부채 관리 의도 vs 10~15억대 매물 거래 위축
- **10.15 대책** (스트레스 DSR +3.0%p): 대출 억제 의도 vs 매물잠김 심화
- **토지거래허가구역 확대**: 투기 억제 의도 vs 비규제 인접지역 풍선효과

**2. 정책 수혜 지역 분석**
- 6억 대출 활용 가능 지역: 10~12억대 매물 (6억 대출 + 4~6억 자금)
- 비규제지역 중 유망 지역: 경기 외곽, 인천 일부
- 풍선효과 수혜 지역: 규제 인접 비규제 지역

**3. 정책 주의 섹터 분석**
- 초고가 주택 (25억 초과): 주담대 2억 제한
- 다주택 투자: 취득세 8~12%, 종부세 중과
- 재건축/재개발: 이주비 대출 제한 영향

**4. 투자자 유형별 전략**
| 투자자 유형 | 현재 정책에서의 유불리 | 추천 전략 |
|------------|---------------------|----------|
| 무주택 실수요자 | ✅ 유리 (정책대출, 생애최초) | 신생아 특례 활용 |
| 1주택 갈아타기 | ⚠️ 중립 | 일시적 2주택 활용 |
| 다주택 투자자 | ❌ 불리 | 중과 유예 기간 내 정리 |
| 법인 투자자 | ❌ 불리 | 주거용 취득 자제 |

**5. 정책 변화 모니터링 포인트**
- 금리 추가 인하 시: 대출 여력 증가 → 거래량 회복
- DSR 규제 완화 시: 매물잠김 해소 → 거래 활성화
- 다주택 중과 유예 종료 (2026.05): 매물 출회 가능성
