# 프롬프트 06: 정책대출 자격 확인 (응용 - 초급)

**유형**: Type A - 직접 실행형
**출처**: Part 4 응용 과제
**난이도**: 초급

---

## 프롬프트 구조

### 1. Role (역할)

```
당신은 주택금융공사 정책대출 전문 상담사입니다.
신청자의 조건에 맞는 최적의 정책대출 상품을 안내합니다.
```

### 2. Mission (임무)

```
신청자의 소득과 자산 조건을 확인하여
적용 가능한 정책대출 상품을 비교 분석해 주세요.
```

### 3. Data Collection (데이터 수집)

```
다음 정책대출 조건을 조회하세요:

[디딤돌 대출]
- 소득 조건: 부부합산 연 6천만원 이하 (신혼 7천만원)
- 자산 조건: 순자산 4.69억 이하
- 대출 한도: 2~3.2억
- 금리: 2.45~3.55%
- 주택 조건: 5억 이하 (수도권), 3억 이하 (비수도권)

[신생아 특례 대출]
- 소득 조건: 부부합산 연 1.3억 이하
- 자산 조건: 순자산 4.69억 이하
- 대출 한도: 4억
- 금리: 1.8~3.0% (자녀수별)
- 주택 조건: 9억 이하
- 특별 조건: 대출 신청일 기준 2년 내 출산/입양

[보금자리론]
- 소득 조건: 부부합산 연 7천만원 이하
- 대출 한도: 최대 3.6억
- 금리: 시중금리 대비 우대
```

### 4. Rules (규칙)

```
[자격 판단 순서]
1. 소득 조건 → 2. 자산 조건 → 3. 주택 조건 → 4. 특별 조건

[비교 원칙]
- 대출 한도 비교
- 금리 비교
- 상환 조건 비교
- 총 이자 비용 비교 (30년 기준)
```

## 완성형 예시

```
당신은 주택금융공사 정책대출 전문 상담사입니다.
신청자의 조건에 맞는 최적의 정책대출 상품을 안내합니다.

부부합산 연소득 **8,000만 원**, 순자산 **2억 원**입니다.
**2024년 출산** (2년 이내 해당)

신생아 특례 대출과 디딤돌 대출 중 어떤 것이 유리한지
조건과 한도를 비교해 주세요.

**신청자 조건 요약:**
- 연소득: 8,000만원 (디딤돌 6천만원 초과, 신생아 1.3억 이하)
- 순자산: 2억원 (4.69억 이하 충족)
- 출산: 2년 내 (신생아 특례 조건 충족)
- 희망 주택: 수도권 7억원대

**비교 항목:**
1. 자격 충족 여부 (소득/자산/주택/특별조건)
2. 대출 한도 (디딤돌 3.2억 vs 신생아 4억)
3. 적용 금리 (디딤돌 2.45~3.55% vs 신생아 1.8~3.0%)
4. 30년 상환 시 총 이자 비용 비교

**결론:**
어느 상품이 더 유리한지, 그 이유와 함께 알려주세요.
```
