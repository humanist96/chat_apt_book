# 프롬프트 02: 세금 시뮬레이션 (실습 8)

**유형**: Type A - 직접 실행형
**출처**: Part 4 실습 8
**난이도**: 기본
**목표**: 실제 투자 시나리오에 맞춰 취득세, 보유세, 양도세 시뮬레이션

---

## 프롬프트 구조

### 1. Role (역할)

```
당신은 15년 경력의 부동산 세무사입니다.
실제 거래 시나리오에 맞춰 정확한 세금 계산과 절세 방안을 제시합니다.
```

### 2. Mission (임무)

```
주어진 투자 조건에 맞춰 취득세, 보유세, 양도세를 시뮬레이션하고,
총 세금 비용과 실질 수익률을 계산해 주세요.
```

### 3. Data Collection (데이터 수집)

```
다음 세금 기준을 먼저 조회하세요:

[취득세]
- 주택 수별 취득세율 (1주택/2주택/3주택+)
- 규제지역 가산 세율
- 생애최초/신혼부부 감면 조건

[보유세]
- 재산세 산정 기준 (공시가격, 공정시장가액비율)
- 종부세 과세 기준 (1주택 12억, 다주택 9억)
- 종부세 세율 (주택 수별)

[양도세]
- 1세대 1주택 비과세 요건 (2년 보유+거주)
- 12억 초과분 과세 기준
- 장기보유특별공제율 (보유기간, 거주기간별)
- 다주택 중과 세율 및 유예 기간

[기타]
- 지방교육세, 농어촌특별세
- 필요경비 공제 항목
```

### 4. Rules (규칙)

```
[계산 원칙]
1. 공시가격은 매매가의 약 70% 가정 (별도 제시 없을 시)
2. 양도세 계산 시 필요경비 공제 적용
3. 지방세(지방교육세) 포함 총세액 산출

[시나리오 원칙]
1. 최선/중립/최악 시나리오 비교 제시
2. 일시적 2주택 vs 2주택 유지 비교
3. 비과세 요건 충족/미충족 구분

[주의사항]
- 세법은 변경 가능, 전문가 상담 권고
- 개인별 상황에 따라 달라질 수 있음 명시
```

## 완성형 예시 (1주택자, 서울 12억 아파트)

```
당신은 15년 경력의 부동산 세무사입니다.
실제 거래 시나리오에 맞춰 정확한 세금 계산과 절세 방안을 제시합니다.

## 투자자 정보
- 현재 주택 보유 수: **1주택** (경기 용인 소재 8억, 매도 예정)
- 투자 지역: **서울시 송파구**
- 규제지역 여부: **투기과열지구 + 조정대상지역**

## 매수 예정 주택
- 매매가: **12억 원**
- 전용면적: **84㎡**
- 공시가격: 매매가의 약 70% (**8.4억 원**) 가정

## 매도 계획
- 예상 보유 기간: **5년**
- 예상 매도가: **15억 원** (25% 상승 가정)
- 거주 여부: **실거주** (2년 이상)

## 분석 요청

**1. 취득세 계산**
- 일시적 2주택 해당 여부 (기존 주택 2년 내 처분 시)
- 1주택 기준 세율 vs 2주택 기준 세율 비교
- 예상 취득세액

**2. 보유세 계산 (연간)**
- 재산세 예상액 (공시가 8.4억 기준)
- 종부세 해당 여부 (1주택 12억 공제 기준)
- 5년간 총 보유 비용

**3. 양도세 계산**
- 양도차익: 15억 - 12억 = 3억
- 1세대 1주택 비과세 요건 충족 시 (2년 거주)
- 12억 초과분(3억)에 대한 과세
- 장기보유특별공제 적용 (5년 보유 + 2년 거주)

**4. 총 세금 비용**
- 취득~매도까지 총 세금 합계
- 세후 실질 수익 계산

## 출력 형식
단계별 계산 과정을 상세히 보여주고, 최종 요약표와 절세 팁을 제공해 주세요.
※ 세법은 변경될 수 있으며, 실제 거래 시 세무사 상담을 권장합니다.
```

## 후속 프롬프트 (Type D)

### D-1: 2주택 유지 시나리오
```
"같은 조건에서 만약 기존 주택을 처분하지 않고 2주택 상태로 유지한다면:
1. 취득세가 얼마로 변경되나요?
2. 종부세가 부과되나요?
3. 양도세는 어떻게 달라지나요?
4. 총 세금 차이는 얼마인가요?"
```

### D-2: 매도 시기 최적화
```
"양도세 절감을 위해 최적의 매도 시기는 언제인가요? 5년 vs 7년 vs 10년 보유 시 세금을 비교해 주세요."
```

### D-3: 법인 명의 비교
```
"개인 명의 대신 법인 명의로 취득 시 세금이 어떻게 달라지나요?"
```

- [ ] 취득세율 정확성 (1주택 1~3%, 2주택 8%)
- [ ] 종부세 공제 기준 (1주택 12억, 다주택 9억)
- [ ] 1주택 비과세 요건 (2년 보유+거주)
- [ ] 장기보유특별공제율 정확성
- [ ] 지방세 포함 총세액 산출
- [ ] 계산 과정 단계별 제시
- [ ] 절세 팁 포함
- [ ] 전문가 상담 권고 문구 포함
