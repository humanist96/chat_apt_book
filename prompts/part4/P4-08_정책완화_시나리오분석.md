# 프롬프트 08: 정책 완화 시나리오 분석 (응용 - 중급)

**유형**: Type A - 직접 실행형
**출처**: Part 4 응용 과제
**난이도**: 중급

---

## 프롬프트 구조

### 1. Role (역할)

```
당신은 15년 경력의 부동산 정책 분석 전문가입니다.
정책 변화 시나리오가 시장에 미치는 영향을 예측합니다.
```

### 2. Mission (임무)

```
가정된 정책 완화 시나리오가 시행될 경우
시장 영향과 수혜 지역을 분석해 주세요.
```

### 3. Data Collection (데이터 수집)

```
다음 현행 규제와 완화 시나리오를 비교하세요:

[현행 규제]
- LTV: 40% (규제지역)
- 스트레스 DSR: +3.0%p (규제지역) / +1.5%p (비규제지역)
- 다주택 양도세: 중과 유예 중 (2026.05.09까지)

[완화 시나리오]
- LTV: 50%로 상향 (가정)
- DSR 가산금리: +1.5%p로 완화 (현행 +3.0%p에서 절반 완화 가정)
- 다주택 양도세: 중과 폐지 (가정)

[과거 사례]
- 2022년 이전 LTV 70% 시기 시장 반응
- 정책 완화 후 가격 변동 사례
```

### 4. Rules (규칙)

```
[분석 원칙]
1. 각 완화 조치별 독립 분석
2. 복합 효과 분석 (동시 시행 시)
3. 지역별 차등 영향 분석

[수혜 지역 판단 기준]
- 대출 여력 증가 효과 큰 지역
- 매물잠김 해소 효과 큰 지역
- 거래량 회복 가능성 높은 지역

[주의사항]
- 가정된 시나리오임을 명시
- 과도한 확신 표현 자제
- 리스크 요인 함께 제시
```

### 5. Output Format (출력 형식)


## 완성형 예시

```
당신은 15년 경력의 부동산 정책 분석 전문가입니다.
정책 변화 시나리오가 시장에 미치는 영향을 예측합니다.

만약 **2026년**에 다음 정책 완화가 시행된다면:
- **LTV 50%로 상향** (현행 40%)
- **DSR 가산금리 1.5%로 완화** (현행 규제지역 +3.0%p → +1.0%p 가정)
- **다주택 양도세 중과 폐지** (현행 유예 → 완전 폐지)

다음을 분석해 주세요:

**1. 각 완화 조치별 시장 영향**
- LTV 50%: 대출 가능액 증가 효과 (예: 10억 주택 4억→5억)
- DSR 완화: 매물잠김 해소 효과
- 양도세 완화: 다주택자 매물 출회 가능성

**2. 수혜 지역 TOP 3**
- 대출 한도 증가 수혜 지역
- 매물잠김 해소 수혜 지역
- 거래 활성화 기대 지역

**3. 복합 효과 (3가지 동시 시행 시)**
- 시장 전체 영향 규모
- 과열 리스크 여부

**4. 투자 전략 제안**
- 완화 전 준비 전략
- 완화 후 대응 전략

※ 본 분석은 가정된 시나리오이며, 실제 정책 변화와 다를 수 있습니다.
```
