# 프롬프트 09: 5년 투자 로드맵 설계 (응용 - 고급)

**유형**: Type A - 직접 실행형
**출처**: Part 4 응용 과제
**난이도**: 고급

---

## 프롬프트 구조

### 1. Role (역할)

```
당신은 20년 경력의 부동산 자산관리 전문가입니다.
장기 투자 전략을 설계하고 단계별 실행 계획을 수립합니다.
```

### 2. Mission (임무)

```
투자자의 현재 상황과 목표를 기반으로
5년간 단계별 투자 로드맵을 설계해 주세요.
```

### 3. Data Collection (데이터 수집)

```
다음 데이터를 조회하세요:

[정책대출 현황]
- 디딤돌 대출: 소득 6천만원, 한도 2~3.2억
- 신생아 특례: 소득 1.3억, 한도 4억
- 보금자리론: 소득 7천만원, 한도 3.6억

[대출 규제]
- LTV 40%, DTI 40%
- 스트레스 DSR: 규제지역 +3.0%p
- 주담대 한도: 가격대별 제한

[세금 규제]
- 취득세: 1주택 1~3%, 2주택 8%
- 양도세: 1주택 비과세 (12억 이하), 다주택 중과 유예
- 종부세: 1주택 12억 공제

[시장 데이터]
- 지역별 평균 매매가 (KB부동산)
- 지역별 전세가율
- 5년 평균 상승률 (참고용)
```

### 4. Rules (규칙)

```
[로드맵 설계 원칙]
1. 단계별 실현 가능성 검증
2. 대출/세금 규제 준수
3. 보수적 시나리오 기반 (낙관적 가정 경계)
4. 리스크 요인 명시

[단계 구성]
- 1단계 (0~2년): 첫 주택 매수
- 2단계 (2~4년): 갈아타기 또는 추가 매수
- 3단계 (4~5년): 포트폴리오 최적화

[목표 검증]
- 현실적 도달 가능성 평가
- 필요 수익률 역산
- 리스크 시나리오별 결과
```

## 플레이스홀더 버전

```
당신은 20년 경력의 부동산 자산관리 전문가입니다.
장기 투자 전략을 설계하고 단계별 실행 계획을 수립합니다.

현재 [주택 보유 상황], 예산 [금액]억(대출 포함), 연소득 [금액]만원입니다.

다음 조건으로 5년간 투자 로드맵을 설계해 주세요:
- 1단계: 첫 주택 매수 (정책대출 활용)
- 2단계: 갈아타기 또는 추가 매수
- 3단계: 최종 자산 목표 [금액]억

각 단계별로 다음을 포함해 주세요:
- 추천 지역 및 가격대

- 세금 영향 (취득세, 양도세)
- 주의사항 및 리스크

시나리오별(낙관/중립/보수적) 결과도 제시해 주세요.
```

## 완성형 예시

```
당신은 20년 경력의 부동산 자산관리 전문가입니다.
장기 투자 전략을 설계하고 단계별 실행 계획을 수립합니다.

현재 **무주택**, 예산 **5억**(대출 포함), 연소득 **7천만원**입니다.
자기자본 약 **2억원** 보유 중입니다.

다음 조건으로 5년간 투자 로드맵을 설계해 주세요:

**1단계 (0~2년): 첫 주택 매수**
- 정책대출 활용 (디딤돌/신생아 특례 조건 검토)
- 예산 5억 내 추천 지역
- 전세가율 높은 지역 우선

**2단계 (2~4년): 갈아타기 또는 추가 매수**
- 1주택 비과세 요건 충족 후 갈아타기
- 또는 일시적 2주택 활용

**3단계 (4~5년): 최종 자산 목표 15억**
- 포트폴리오 최적화
- 레버리지 수준 조정

각 단계별 포함 항목:
- 추천 지역 및 가격대 (구체적으로)
- 대출 전략 (정책대출 한도, 주담대 활용)
- 세금 영향 (취득세, 양도세 시뮬레이션)
- 주의사항 및 체크포인트

시나리오별 결과:
- 낙관적 (연 10% 상승)
- 중립적 (연 5% 상승)
- 보수적 (연 2% 상승)
- 최악 (연 3% 하락)

⚠️ 본 로드맵은 참고용이며 투자 결과를 보장하지 않습니다.
```

## 후속 프롬프트 (Type D)

### D-1: 단계별 상세화
```
"1단계 첫 주택 매수를 더 상세히 분석해 주세요. 구체적인 단지 3곳을 추천해 주세요."
```

### D-2: 예산 조정
```
"예산을 7억으로 늘리면 로드맵이 어떻게 변하나요?"
```

### D-3: 위험 관리
```
"최악의 시나리오(3년 연속 하락)에서 손실을 최소화하는 방법은?"
```

### D-4: 대출 상세
```
"각 단계별 대출 상환 계획과 원리금 부담을 계산해 주세요."
```
