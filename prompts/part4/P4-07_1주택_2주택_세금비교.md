# 프롬프트 07: 1주택 vs 2주택 세금 비교 (응용 - 중급)

**유형**: Type A - 직접 실행형
**출처**: Part 4 응용 과제
**난이도**: 중급

---

## 프롬프트 구조

### 1. Role (역할)

```
당신은 15년 경력의 부동산 세무사입니다.
다주택 보유 시나리오별 세금 영향을 정확히 분석합니다.
```

### 2. Mission (임무)

```
기존 주택 보유자가 추가 매수 시 1주택 유지(처분)와
2주택 유지 시나리오의 세금 차이를 비교해 주세요.
```

### 3. Data Collection (데이터 수집)

```
다음 세금 기준을 조회하세요:

[일시적 2주택]
- 정의: 신규 주택 취득 후 기존 주택 2년 내 처분
- 취득세: 1주택 세율 적용 (1~3%)
- 양도세: 기존 주택 비과세 적용 가능

[2주택 유지]
- 취득세: 조정대상지역 8%, 비조정 1~3%
- 종부세: 9억 공제 (합산), 중과세율 적용 가능
- 양도세: 중과(유예 중), 장기보유공제 제한

[주요 변수]
- 기존 주택 위치 (규제/비규제)
- 신규 주택 위치 (규제/비규제)
- 보유 기간 및 거주 여부
```

### 4. Rules (규칙)

```
[비교 원칙]
1. 동일 조건에서 시나리오 A vs B 직접 비교
2. 취득세, 종부세, 양도세 각각 산출
3. 총 세금 차이 금액화

[손익분기점 분석]
- 2주택 유지가 유리해지는 조건
- 시장 상승률에 따른 변화
```

### 5. Output Format (출력 형식)

```
## 1주택 vs 2주택 세금 시뮬레이션

### 조건 요약
| 항목 | 기존 주택 | 신규 주택 |
|------|----------|----------|
| 위치 | | |
| 매매가 | | |
| 규제지역 | | |

### 시나리오 비교
| 항목 | A: 일시적 2주택 | B: 2주택 유지 | 차이 |
|------|---------------|--------------|------|
| 취득세 | | | |
| 종부세 (X년) | | | |
| 양도세 (신규) | | | |
| **총 세금** | | | |

### 손익분기점 분석
- 2주택 유지 유리 조건: [조건]
- 권고 사항: [제안]
```

```
당신은 15년 경력의 부동산 세무사입니다.
다주택 보유 시나리오별 세금 영향을 정확히 분석합니다.

현재 [지역]에 [금액]억 아파트를 보유 중입니다.

**시나리오 A**: 기존 주택 2년 내 처분 (일시적 2주택)
**시나리오 B**: 2주택 유지

각 시나리오별 세금(취득세/종부세/양도세)을 비교해 주세요.
- 보유 기간: [기간]년 가정
- 예상 매도가: [상승률]% 상승 가정

어떤 시나리오가 유리한지, 손익분기점은 어디인지 분석해 주세요.
```

## 완성형 예시

```
당신은 15년 경력의 부동산 세무사입니다.
다주택 보유 시나리오별 세금 영향을 정확히 분석합니다.

현재 **경기도 용인시**에 **8억** 아파트를 보유 중입니다.
**서울 송파구**에 **12억** 아파트를 추가 매수 시:

**시나리오 A**: 기존 용인 주택 2년 내 처분 (일시적 2주택)
- 취득세: 1주택 기준 (1~3%)
- 종부세: 1주택 12억 공제 적용
- 양도세(신규): 1주택 비과세 요건 충족 가능

**시나리오 B**: 2주택 유지
- 취득세: 2주택 기준 (8%, 조정대상지역)
- 종부세: 합산 과세 (9억 공제)
- 양도세: 중과 유예 중 (2026.05.09까지)

**조건:**
- 보유 기간: 5년
- 신규 주택 실거주: 2년 이상
- 예상 매도가: 서울 15억 (25% 상승)

각 시나리오별 세금(취득세/종부세 5년분/양도세)을 비교하고,
어떤 시나리오가 유리한지 분석해 주세요.
손익분기점(2주택 유지가 유리해지는 조건)도 제시해 주세요.
```

## 후속 프롬프트 (Type D)

### D-1: 처분 타이밍
```
"시나리오 A에서 기존 주택을 1년 내 vs 2년 내 처분 시 세금 차이가 있나요?"
```

### D-2: 중과 유예 종료 후
```
"2026년 5월 이후 다주택 중과가 적용되면 세금이 얼마나 증가하나요?"
```

### D-3: 3주택 시나리오
```
"추가로 1채를 더 매수해 3주택이 되면 세금이 어떻게 변하나요?"
```

## 검증 체크리스트

- [ ] 일시적 2주택 정의 정확성 (2년 내 처분)
- [ ] 2주택 취득세율 (규제지역 8%)
- [ ] 종부세 합산 과세 및 공제 기준
- [ ] 다주택 양도세 중과 유예 기간
- [ ] 시나리오별 세금 명확히 비교
- [ ] 손익분기점 분석 포함
- [ ] 전문가 상담 권고 문구
