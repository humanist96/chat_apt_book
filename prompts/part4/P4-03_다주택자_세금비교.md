# 프롬프트 03: 다주택자 시나리오 세금 비교 (실습 8 후속)

**유형**: Type D - 후속 프롬프트
**출처**: Part 4 실습 8 후속
**난이도**: 기본
**목표**: 1주택 vs 2주택 유지 시 세금 차이 비교

---

## 프롬프트 구조

### 1. Role (역할)

```
당신은 15년 경력의 부동산 세무사입니다.
다주택 보유 시나리오별 세금 차이를 정확히 비교 분석합니다.
```

### 2. Mission (임무)

```
앞서 시뮬레이션한 조건에서 기존 주택을 처분하지 않고
2주택 상태로 유지할 경우의 세금 변화를 분석해 주세요.
```

### 3. Data Collection (데이터 수집)

```
다음 다주택자 세금 기준을 조회하세요:

[취득세 - 2주택]
- 조정대상지역 2주택 취득세율
- 비조정대상지역 2주택 취득세율

[종부세 - 다주택]
- 다주택자 공제 기준 (9억)
- 다주택자 세율 (일반 vs 중과)

[양도세 - 다주택]
- 다주택자 양도세 중과세율
- 중과 유예 기간 (2026.05.09까지)
- 장기보유특별공제 적용 제한
```

### 4. Rules (규칙)

```
[비교 원칙]
1. 동일 조건에서 1주택 vs 2주택 직접 비교
2. 취득세, 보유세, 양도세 각각 비교
3. 총 세금 차이 금액 명시

[시나리오 구분]
- 시나리오 A: 일시적 2주택 (기존 주택 2년 내 처분)
- 시나리오 B: 2주택 유지

[중과 유예 반영]
- 다주택 양도세 중과: 2026.05.09까지 유예
- 유예 종료 후 중과 세율 적용 가정
```

## 플레이스홀더 버전

```
당신은 15년 경력의 부동산 세무사입니다.
다주택 보유 시나리오별 세금 차이를 정확히 비교 분석합니다.

앞서 시뮬레이션한 조건에서 기존 주택을 처분하지 않고
2주택 상태로 유지할 경우의 세금 변화를 분석해 주세요.

## 비교 항목
1. 취득세가 얼마로 변경되나요? (1주택 vs 2주택 세율)
2. 종부세가 부과되나요? (공제 기준 12억 vs 9억)
3. 양도세는 어떻게 달라지나요? (비과세 vs 중과)
4. 총 세금 차이는 얼마인가요?

## 출력 형식
시나리오 A(일시적 2주택) vs 시나리오 B(2주택 유지)를 표로 비교해 주세요.
총 세금 차이와 손익분기점을 분석해 주세요.
```

## 완성형 예시

```
당신은 15년 경력의 부동산 세무사입니다.
다주택 보유 시나리오별 세금 차이를 정확히 비교 분석합니다.

앞서 시뮬레이션한 조건(서울 12억 아파트, 5년 보유, 15억 매도)에서
기존 경기 8억 주택을 처분하지 않고 2주택 상태로 유지할 경우:

## 비교 항목
1. **취득세 변경**: 1주택 1~3% → 2주택 8% (규제지역)
2. **종부세 부과**: 1주택 12억 공제 → 다주택 9억 공제 (합산 과세)
3. **양도세 변경**:
   - 1주택: 12억 초과분만 과세 + 장기보유공제 적용
   - 2주택: 중과(현재 유예 중, 2026.05.09까지) + 장기보유공제 제한
4. **총 세금 차이**: 취득~매도까지 총액 비교

## 추가 분석
- 중과 유예 종료(2026.05.09) 전후 시나리오 비교
- 2주택 유지가 유리해지는 조건 분석

## 출력 형식
시나리오 A(일시적 2주택, 기존 주택 2년 내 처분) vs
시나리오 B(2주택 유지)를 표로 비교해 주세요.
총 세금 차이와 손익분기점을 분석해 주세요.
```

## 후속 프롬프트 (Type D)

### D-1: 처분 타이밍 최적화
```
"2주택 유지 후 어느 시점에 기존 주택을 처분하는 것이 세금상 유리한가요?"
```

### D-2: 3주택 시나리오
```
"만약 추가로 1채를 더 매수해서 3주택이 된다면 세금이 어떻게 변하나요?"
```

### D-3: 증여 vs 매도
```
"기존 주택을 자녀에게 증여하는 것과 매도하는 것 중 어느 쪽이 유리한가요?"
```
