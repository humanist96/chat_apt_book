## 분석 항목

### 1. 교통 개발호재
- GTX 노선 (위 기준 데이터 참조)
- 신규 지하철 노선
- 도로/고속도로 확장
- 개통 시점 (기준 데이터 기준)

### 2. 재개발/재건축
- 진행 중인 정비사업
- 예정 구역
- 예상 분양 시점
- 예상 공급 물량

### 3. 신도시/택지개발
- 주변 신도시 개발
- 입주 예정 물량
- 시너지/경쟁 효과

### 4. 산업단지/일자리
- 신규 기업 이전
- 산업단지 확장
- 예상 고용 규모

## Rules

1. **기준 데이터 우선**: 위 "기준 데이터" 섹션의 개통 연도를 반드시 사용. 검색 결과와 충돌 시 기준 데이터 우선
2. **확정/예정 구분**: 확정 호재와 예정 호재를 명확히 구분
3. **진행 단계 명시**: 재개발은 정비구역지정/조합설립/사업시행인가/관리처분인가/착공/준공 단계 표기
4. **가격 영향 분석**: 개통/준공 전후 예상 시세 영향 포함
5. **데이터 출처**: 국토교통부, 서울특별시/경기도 고시 기준
6. **검증 체크**: GTX-C는 2030년, GTX-A 동탄역은 2024.03 개통 완료로 표기
