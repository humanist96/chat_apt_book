# 프롬프트 05: 전세가율 역사적 비교 (응용 - 초급)

**유형**: Type A - 직접 실행형
**출처**: Part 3 응용 과제
**난이도**: 초급

---

## 프롬프트 구조

### 1. Role (역할)

```
당신은 15년 경력의 부동산 시장 사이클 분석 전문가입니다.
과거 데이터를 기반으로 현재 시장의 위치를 진단합니다.
```

### 2. Mission (임무)

```
서울 아파트 전세가율의 과거 10년간 추이를 분석하고,
현재 수준이 역사적으로 어느 위치인지 평가해 주세요.
```

### 3. Data Collection (데이터 수집)

```
다음 데이터를 먼저 조회하세요:

[전세가율 시계열]
- 서울 전세가율 월별 데이터 (KB부동산, 2015~현재)
- 연도별 최고점/최저점
- 현재 전세가율 (가장 최근 발표 기준)

[주요 시점]
- 2018년 전세가율 (저점 시기)
- 2021년 전세가율 (고점 시기)
- 50% 이하 기록 여부 및 시점

[배경 정보]
- 각 시기별 정책 환경 (규제, 금리)
- 시장 상황 (상승장/하락장/횡보)
```

### 4. Rules (규칙)

```
[분석 기준]
1. 전세가율 70% 이상: 투자 매력도 높음 (갭투자 유리)
2. 전세가율 60~70%: 중립 구간
3. 전세가율 50~60%: 투자 리스크 존재
4. 전세가율 50% 이하: 역사적 저점 구간, 장기 관점 매수 검토

[역사적 맥락]
- 전세가율 50% 이하: 과거 금융위기(2008) 수준
- 전세가율 급락 → 보통 1-2년 내 반등 경향
- 전세가율과 매매가격은 6-12개월 시차 존재
```

### 5. Output Format (출력 형식)

```

### 1. 10년간 전세가율 추이
| 연도 | 평균 전세가율 | 최고점 | 최저점 | 시장 상황 |
|------|-------------|--------|--------|----------|
| 2015 | XX.X% | | | |
| ... | | | | |
| 2025 | XX.X% | | | |

### 2. 역사적 주요 구간
| 구간 | 전세가율 | 시장 상황 | 정책 환경 |
|------|----------|----------|----------|
| 2018년 저점 | XX% | | |
| 2021년 고점 | XX% | | |
| 현재 | XX% | | |

### 3. 50% 이하 기록 분석
- 기록 여부: [예/아니오]
- 기록 시점: [날짜]

- 이후 추이: [설명]

### 4. 현재 위치 평가
- 10년 평균 대비: [상/중/하]
- 역사적 백분위: 하위 XX%
- 시장 사이클: [회복기/상승기/과열기/하락기/침체기]

- [구체적 시사점 2-3개]
```

## 완성형 예시

```
당신은 15년 경력의 부동산 시장 사이클 분석 전문가입니다.
과거 데이터를 기반으로 현재 시장의 위치를 진단합니다.

서울 아파트 전세가율의 과거 10년간 추이를 분석해 주세요.
- 현재 전세가율 **51.27%** (KB부동산, 2025.11 기준)는 역사적으로 어느 수준인가요?
- 전세가율이 50% 이하로 떨어진 적이 있나요?

다음 데이터를 조회하세요:
- 서울 전세가율 연도별 데이터 (KB부동산, 2015~현재)
- 2018년(저점 약 68%), 2021년(고점 약 57%) 전세가율
- 금융위기(2008년) 당시 전세가율

출력 형식: 10년 추이 표, 주요 구간 분석, 50% 이하 기록 여부, 현재 위치 평가, 투자 시사점
```

```
"서울 전체 전세가율과 강남3구, 노도강(노원/도봉/강북) 전세가율을 비교해 주세요."
```

### D-2: 반등 시나리오
```
"과거 전세가율 저점에서 반등한 사례를 분석하고, 현재 시점 반등 가능성을 예측해 주세요."
```

### D-3: 투자 전략 연계
```
"현재 전세가율 수준에서 갭투자 vs 실거주 투자 중 어느 쪽이 유리한지 분석해 주세요."
```
