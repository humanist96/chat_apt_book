# 프롬프트 08: 종합 투자 지역 선정 (응용 - 고급)

**유형**: Type A - 직접 실행형
**출처**: Part 3 응용 과제
**난이도**: 고급

---

## 프롬프트 구조

### 1. Role (역할)

```
당신은 20년 경력의 부동산 포트폴리오 매니저입니다.
다중 조건을 종합 분석하여 최적의 투자 지역을 선정합니다.
```

### 2. Mission (임무)

```
다음 조건을 모두 만족하는 투자 지역 3곳을 추천하고,
각 지역의 수급/전세가율/인구 데이터를 종합 분석해 주세요.
```

### 3. Data Collection (데이터 수집)

```
다음 조건별 데이터를 조회하세요:

[조건 1: 가격대]
- 수도권 지역별 아파트 평균 매매가 (KB부동산, 최신)
- 수도권 지역별 전세가 (KB부동산, 최신)
- 매매 [예산금액] 이내 지역 필터링

[조건 2: 입주물량]
- 2026년 지역별 입주예정물량 (부동산R114)
- 2025년 대비 증감률
- 5년 평균 대비 수준

[조건 3: 인구]
- 지역별 최근 3년 순이동 (통계청)
- 순유입 지역 필터링
- 주요 유입 연령대 분석

[조건 4: 전세가율]
- 지역별 전세가율 (KB부동산, 최신)
- 55% 이상 지역 필터링
- 전년 대비 변동 추이

[조건 5: 교통]
- 강남역 기준 대중교통 소요시간
- GTX/신분당선 등 예정 노선 반영
- 40분 이내 지역 필터링
```

### 4. Rules (규칙)

```
[조건 우선순위]
1순위: 가격대 (예산 초과 시 제외)
2순위: 교통 (40분 초과 시 제외)
3순위: 전세가율 (55% 미만 시 감점)
4순위: 입주물량 (2026년 감소 여부)
5순위: 인구 (3년 순유입 여부)

[점수 체계 (100점 만점)]
- 가격 적정성: 20점 (예산 대비 여유 있을수록 가점)
- 교통 접근성: 25점 (소요시간 짧을수록 고점)
- 전세가율: 20점 (높을수록 고점, 55% 기준 시작)
- 입주물량: 20점 (감소폭 클수록 고점)
- 인구 유입: 15점 (유입 규모 클수록 고점)

[최종 선정 기준]
- 5개 조건 중 4개 이상 충족
- 종합 점수 70점 이상
- 명확한 리스크 요인 없음

[현행 규제 반영]
- LTV 40%, DTI 40% (2025.10.16 기준)
- 스트레스 DSR: 규제지역 +3.0%p / 비규제지역 +1.5%p / 지방 +0.75%p
- 규제지역 여부 확인 필수
```

```
당신은 20년 경력의 부동산 포트폴리오 매니저입니다.
다중 조건을 종합 분석하여 최적의 투자 지역을 선정합니다.

다음 조건을 모두 만족하는 투자 지역 3곳을 추천해 주세요:
1. 예산: 매매 [매매예산]억 이내, 전세 [전세예산]억 이내
2. 입주물량: 2026년 감소 예정
3. 인구: 최근 3년 순유입
4. 전세가율: [전세가율]% 이상
5. 교통: 강남 [소요시간]분 이내

각 지역의 수급/전세가율/인구 데이터를 종합 분석해 주세요.

조회 데이터:
- 지역별 매매가/전세가 (KB부동산)
- 2026년 입주예정물량 (부동산R114)
- 3년 인구 순이동 (통계청)

출력 형식: 조건 요약 → 스크리닝 결과 → TOP 3 상세 분석 → 실행 로드맵
```

## 완성형 예시

```
당신은 20년 경력의 부동산 포트폴리오 매니저입니다.
다중 조건을 종합 분석하여 최적의 투자 지역을 선정합니다.

다음 조건을 모두 만족하는 투자 지역 3곳을 추천해 주세요:
1. 예산: 매매 **10억** 이내, 전세 **6억** 이내
2. 입주물량: 2026년 감소 예정
3. 인구: 최근 3년 순유입
4. 전세가율: **55%** 이상
5. 교통: 강남 **40분** 이내

각 지역의 수급/전세가율/인구 데이터를 종합 분석해 주세요.

조회 데이터:
- 수도권 지역별 아파트 평균 매매가/전세가 (KB부동산, 최신)
- 2026년 지역별 입주예정물량 (부동산R114)
- 지역별 최근 3년 인구 순이동 (통계청)
- 지역별 전세가율 (KB부동산, 최신)
- 강남역 기준 대중교통 소요시간 (카카오맵/네이버지도)

현행 규제 반영:
- LTV 40%, DTI 40% (2025.10.16 기준)
- 스트레스 DSR: 규제지역 +3.0%p / 비규제지역 +1.5%p / 지방 +0.75%p

점수 체계 (100점 만점):
- 가격 적정성 20점 (예산 대비 여유)
- 교통 접근성 25점 (소요시간)
- 전세가율 20점 (55% 기준)
- 입주물량 20점 (감소폭)
- 인구 유입 15점 (유입 규모)

출력 형식:
1. 조건 요약 표
2. 조건 충족 지역 스크리닝 (✅/❌ 표기)
3. TOP 3 상세 분석 (점수표 + 투자 포인트 + 리스크 + 추천 단지)
4. 투자 실행 로드맵
5. 주의사항 (법적 고지)
```

## 후속 프롬프트 (Type D)

### D-1: 단지 추천 상세화
```
"1위 지역 내에서 조건에 맞는 구체적인 아파트 단지 5곳을 추천해 주세요."
```

### D-2: 예산 조정 시나리오
```
"예산을 8억으로 낮추면 어떤 지역이 추천되나요? TOP 3를 다시 분석해 주세요."
```

### D-3: 리스크 심화 분석
```
"추천 지역별 최악의 시나리오(가격 하락, 공실 리스크)를 분석해 주세요."
```

### D-4: 대출 시뮬레이션
```
"1위 지역 10억 매물 매수 시 필요한 자기자본과 대출 한도를 계산해 주세요."
```

- [ ] 현행 규제 (LTV/DTI/DSR) 반영
- [ ] 법적 고지사항 포함
