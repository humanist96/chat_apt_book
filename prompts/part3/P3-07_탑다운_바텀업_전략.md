# 프롬프트 07: 탑다운 vs 바텀업 전략 분석 (응용 - 중급)

**유형**: Type A - 직접 실행형
**출처**: Part 3 응용 과제
**난이도**: 중급

---

## 프롬프트 구조

### 1. Role (역할)

```
당신은 15년 경력의 부동산 투자 전략 컨설턴트입니다.
시장 환경에 따른 최적의 투자 접근법을 제시합니다.
```

### 2. Mission (임무)

```
현재 시장에서 탑다운 전략과 바텀업 전략 중 어느 쪽이 유효한지 분석하고,
각 전략의 수혜 지역을 3곳씩 제시해 주세요.
```

### 3. Data Collection (데이터 수집)

```
다음 데이터를 먼저 조회하세요:

[시장 환경 진단]
- 서울/수도권 매매가 변동률 (최근 6개월, 한국부동산원)
- 거래량 추이 (전년 대비, 한국부동산원)
- 전세가율 수준 (KB부동산, 최신)
- 금리 환경 (한국은행 기준금리, 주담대 금리)

[탑다운 지표]
- 강남권 가격 상승률 vs 외곽 상승률
- 신고가 갱신 단지 수 (서울 기준)
- 규제지역 현황 (투기과열지구, 조정대상지역)

[바텀업 지표]
- 지역별 미분양 현황 (국토부)
- 입주물량 감소 예정 지역
- 전세가율 70% 이상 지역
- 호재(교통, 개발) 예정 지역
```

### 4. Rules (규칙)

```
[탑다운 전략]
정의: 시장 전체 상승에 베팅, 핵심 지역 우선 투자
유효 시점:
- 금리 인하 사이클
- 강남권 선도 상승 확인
- 거래량 증가 추세
- 정책 완화 기대

적합 지역:
- 강남권, 마용성, 목동 등 서울 핵심
- 분당, 과천, 광교 등 1기 신도시 핵심
- GTX/교통 호재 역세권

[바텀업 전략]
정의: 개별 가치주 발굴, 저평가 지역 선별 투자
유효 시점:
- 금리 고점 횡보
- 시장 전체 박스권
- 지역별 차별화 심화
- 공급 감소 지역 발생

적합 지역:
- 전세가율 70% 이상 + 입주물량 감소
- 교통 호재 확정 + 가격 미반영
- 재개발/재건축 사업 진행 지역
- 미분양 소진 완료 지역

[전략 선택 기준]
- 강남 선도 상승 → 탑다운 유효
- 강남 정체 + 외곽 차별화 → 바텀업 유효
- 혼조 → 양 전략 병행 (7:3 또는 5:5)
```

## 플레이스홀더 버전

```
당신은 15년 경력의 부동산 투자 전략 컨설턴트입니다.

현재 시장에서 탑다운 전략과 바텀업 전략 중 어느 쪽이 유효한지 분석해 주세요.

다음 데이터를 조회하세요:

- 거래량 추이 (전년 대비)
- 전세가율, 금리 환경

출력 형식: 시장 환경 표, 전략 유효성 평가, 수혜 지역 3곳씩 (이유+리스크 포함)
```

## 완성형 예시

```
당신은 15년 경력의 부동산 투자 전략 컨설턴트입니다.
시장 환경에 따른 최적의 투자 접근법을 제시합니다.

현재(**2025년 12월**) 시장에서 탑다운 전략과 바텀업 전략 중 어느 쪽이 유효한지 분석해 주세요.
각 전략의 수혜 지역을 3곳씩 제시해 주세요.

다음 데이터를 조회하세요:
- 서울/수도권 매매가 변동률 (최근 6개월, 한국부동산원)
- 거래량 추이 (2024년 동기 대비)
- 전세가율 **51.27%** (KB부동산, 2025.11 기준)
- 기준금리 환경 (한국은행)
- 강남3구 vs 노도강(노원/도봉/강북) 상승률 비교

전략 선택 기준:
- 탑다운: 강남 선도 상승, 금리 인하 사이클, 거래량 증가
- 바텀업: 시장 박스권, 지역별 차별화 심화, 개별 호재 지역

출력 형식:
1. 시장 환경 진단 표
2. 전략 유효성 평가 (⭐ 5점 만점)
3. 탑다운 수혜 지역 3곳 (선정 이유 + 리스크)
4. 바텀업 수혜 지역 3곳 (핵심 호재 + 리스크)
5. 투자 실행 가이드 (단기/중기/장기)
```
