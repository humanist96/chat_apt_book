# 프롬프트 06: 가격 흐름 패턴 분석 (응용 - 중급)

**유형**: Type A - 직접 실행형
**출처**: Part 3 응용 과제
**난이도**: 중급

---

## 프롬프트 구조

### 1. Role (역할)

```
당신은 15년 경력의 부동산 시장 사이클 분석 전문가입니다.
가격 확산 패턴을 분석하여 시장 국면을 진단합니다.
```

### 2. Mission (임무)

```
과거 상승장과 하락장의 지역별 가격 확산 순서를 분석하고,
현재 시장이 어느 단계에 위치해 있는지 진단해 주세요.
```

### 3. Data Collection (데이터 수집)

```
다음 데이터를 먼저 조회하세요:

[상승장 분석 (2020-2021)]
- 지역별 가격 상승 시작 시점 (한국부동산원)
- 가격 상승 선도 지역 순서
- 상승률 피크 도달 시점

[하락장 분석 (2022-2023)]
- 지역별 가격 하락 시작 시점
- 가격 하락 선도 지역 순서
- 하락률 저점 도달 시점

[현재 시장 (2024-2025)]
- 서울/수도권/지방 월별 변동률 (최근 6개월)
- 거래량 추이 (한국부동산원)
- 지역별 회복/하락 현황
```

### 4. Rules (규칙)

```
[가격 확산 패턴 - 상승장]
1단계: 강남권 (서초/강남/송파) 선도 상승
2단계: 서울 주요 지역 확산 (용산/마포/성동 등)
3단계: 서울 전역 + 수도권 핵심 (분당/과천)
4단계: 수도권 외곽 + 지방 광역시
5단계: 전국 확산 (마지막 단계)

[가격 확산 패턴 - 하락장]
1단계: 외곽/비핵심 지역 선제 하락
2단계: 수도권 외곽 → 지방 확산
3단계: 서울 외곽 지역 하락
4단계: 서울 주요 지역 조정
5단계: 강남권 조정 (가장 늦음)

[시장 국면 판단]
- 강남권 상승, 외곽 정체 → 회복 초기
- 서울 전역 상승 → 회복 본격화
- 수도권 전역 상승 → 상승기 진입
- 지방까지 확산 → 과열 주의
```

### 5. Output Format (출력 형식)


## 플레이스홀더 버전

```
당신은 15년 경력의 부동산 시장 사이클 분석 전문가입니다.
가격 확산 패턴을 분석하여 시장 국면을 진단합니다.

과거 상승장(2020-2021)과 하락장(2022-2023)의 지역별 가격 확산 순서를 분석해 주세요.
현재([현재 연월]) 시장은 어느 단계에 위치해 있나요?

다음 데이터를 조회하세요:
- 상승장: 지역별 가격 상승 시작 시점 및 순서 (한국부동산원)
- 하락장: 지역별 가격 하락 시작 시점 및 순서
- 현재: 서울/수도권/지방 월별 변동률 (최근 6개월)

분석 기준:
- 상승장: 강남권 → 서울 주요 → 서울 전역 → 수도권 → 전국
- 하락장: 외곽 → 수도권 외곽 → 서울 외곽 → 서울 주요 → 강남권

출력 형식: 상승장/하락장 확산 순서 표, 현재 시장 국면 진단, 향후 전망
```

## 완성형 예시

```
당신은 15년 경력의 부동산 시장 사이클 분석 전문가입니다.
가격 확산 패턴을 분석하여 시장 국면을 진단합니다.

과거 상승장(2020-2021)과 하락장(2022-2023)의 지역별 가격 확산 순서를 분석해 주세요.
현재(**2025년 12월**) 시장은 어느 단계에 위치해 있나요?

다음 데이터를 조회하세요:
- 상승장: 지역별 가격 상승 시작 시점 및 순서 (한국부동산원)
  - 강남3구(서초/강남/송파) 2020년 1분기 상승 시작
  - 마용성(마포/용산/성동) 2020년 2분기 확산
- 하락장: 지역별 가격 하락 시작 시점 및 순서
  - 2022년 하반기 외곽/지방 선제 하락
  - 2023년 서울까지 조정 확산
- 현재: 서울/수도권/지방 월별 변동률 (최근 6개월)

분석 기준:
- 상승장 확산: 강남권 → 서울 주요 → 서울 전역 → 수도권 → 전국
- 하락장 확산: 외곽 → 수도권 외곽 → 서울 외곽 → 서울 주요 → 강남권

출력 형식: 상승장/하락장 확산 순서 표, 현재 시장 국면 진단(5단계 중 몇 단계), 향후 시나리오별 전망
```
