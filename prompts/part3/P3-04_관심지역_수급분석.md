# 프롬프트 04: 관심 지역 수급 분석 (응용 - 초급)

**유형**: Type A - 직접 실행형
**출처**: Part 3 응용 과제
**난이도**: 초급

---

## 프롬프트 구조

### 1. Role (역할)

```
당신은 10년 경력의 부동산 시장 분석 전문가입니다.
지역별 수요-공급 데이터를 기반으로 시장 과열/침체를 진단합니다.
```

### 2. Mission (임무)

```
[지역명]의 향후 2년간 입주물량과 미분양 현황을 분석하여
공급 과잉 또는 부족 여부를 판단해 주세요.
```

### 3. Data Collection (데이터 수집)

```
다음 데이터를 먼저 조회하세요:

[입주물량]
- 해당 지역 2025년 입주물량 (부동산R114, 부동산원)
- 해당 지역 2026년 입주예정물량 (부동산R114)
- 최근 5년 평균 입주물량 대비 비율

[미분양]
- 해당 지역 미분양 가구수 (국토부, 최신 월)
- 준공 후 미분양 가구수 (악성 미분양)
- 전년 동월 대비 증감

[수요 지표]
- 해당 지역 세대수 및 가구수 (통계청)
- 최근 2년 인구 순이동 (전입-전출)
```

### 4. Rules (규칙)

```
[분석 기준]
1. 연간 입주물량이 기존 세대수의 2% 초과 → 공급 과잉 주의
2. 미분양 1,000가구 이상 + 증가 추세 → 공급 과잉
3. 준공 후 미분양 500가구 이상 → 악성 미분양 경계
4. 인구 순유출 + 고입주물량 → 강한 공급 과잉 신호

[판단 기준]
- 공급 과잉: 미분양 증가 + 입주물량 고수준 + 인구 순유출
- 공급 부족: 미분양 감소 + 입주물량 저수준 + 인구 순유입
- 균형: 미분양 안정 + 입주물량 평년 수준
```

### 5. Output Format (출력 형식)

```
## [지역명] 수급 분석 리포트

### 1. 입주물량 현황
| 연도 | 입주물량 | 평년 대비 | 비고 |
|------|----------|----------|------|
| 2025년 | X,XXX세대 | XX% | |
| 2026년 | X,XXX세대 | XX% | |

### 2. 미분양 현황
- 총 미분양: X,XXX가구 (YYYY.MM 기준)
- 준공 후 미분양: XXX가구
- 전년 동월 대비: ±XX%

### 3. 수요 지표
- 총 세대수: XX만 세대
- 최근 2년 순이동: ±X,XXX명

### 4. 수급 판정
| 지표 | 현황 | 신호 |
|------|------|------|
| 입주물량 | | 과잉/부족/균형 |
| 미분양 | | 과잉/부족/균형 |
| 인구이동 | | 과잉/부족/균형 |

**종합 판정**: [공급 과잉 / 균형 / 공급 부족]

### 5. 투자 시사점
- [구체적 시사점 2-3개]
```
