# Part 0. 프롤로그 - 왜 지금 부동산과 AI인가

## "부동산 공부, 도대체 어디서부터 시작해야 해요?"

'6개월 안에 신혼집을 구해야 하는' 정대리(31세)가 스마트폰을 내려놓으며 한숨을 쉬었습니다.

"유튜브 영상만 수십 개 봤는데, 다들 말이 달라요. 어떤 사람은 '지금 사라', 어떤 사람은 '절대 사지 마라'... 누구 말을 믿어야 하는 거예요?"

'서울 자가에 대기업 다녔던' 김부장(53세)이 고개를 끄덕였습니다.

"나도 처음에 그랬어. 상승론자, 하락론자... 다 자기 얘기만 하더라고. 결국 내가 직접 판단해야 하는데, 그 판단의 근거가 없으니까 불안했지."

정대리가 답답한 표정으로 물었습니다.

"회사 선배들한테 물어봐도 다 '나도 모르겠다'래요. 부동산 전문가한테 상담받으려면 돈도 들고... 어떻게 해야 해요?"

'부동산 투자로 월급쟁이 부자된' 송과장(38세)이 웃으며 끼어들었습니다.

"두 분 다 **같은 실수**를 하고 있어요."

정대리와 김부장이 동시에 눈을 크게 떴습니다.

"실수요?"

## "남의 말 믿지 말고, 데이터를 보세요"

송과장이 노트북을 열었습니다.

"부동산 투자에서 가장 위험한 건 **'누구 말을 믿을까'** 고민하는 거예요. 상승론자도, 하락론자도, 유튜버도... 다 자기 이익이 있어요."

정대리가 반문했습니다.

"그럼 아무도 못 믿는다는 거예요?"

송과장이 고개를 저었습니다.

"아니요. **데이터는 믿을 수 있어요.** 실거래가, 전세가율, 입주물량... 이런 숫자는 거짓말을 안 해요."

김부장이 끼어들었습니다.

"데이터라... 근데 그거 보려면 전문가여야 하는 거 아냐?"

송과장이 미소를 지었습니다.

"**10년 전에는 그랬어요.** 하지만 지금은 완전히 달라졌습니다."

## 정보 민주화의 시대

송과장이 화이트보드에 표를 그렸습니다.

"예전과 지금이 어떻게 다른지 보여드릴게요."

| 10년 전 | 지금 |
|---------|------|
| 실거래가는 공인중개사만 알았다 | 호갱노노에서 **3초 만에** 확인 |
| 전세가율 계산은 전문가 영역 | 앱이 **자동으로** 계산해줌 |
| 입주물량 정보는 업계 내부용 | 아실에서 **누구나** 조회 가능 |
| "속는 호갱"이 흔했다 | **"호갱노노"** 앱 이름의 유래 |

정대리가 놀랐습니다.

"저도 호갱노노 쓰긴 하는데... 그냥 시세만 봤어요."

송과장이 고개를 끄덕였습니다.

"대부분 그래요. 앱은 있는데 **어떻게 활용해야 하는지** 모르는 거죠. 그래서 여전히 '누가 뭐래더라'에 휘둘리는 거예요."

김부장이 물었습니다.

"그럼 어떻게 해야 해?"

송과장이 대답했습니다.

"**데이터를 직접 읽는 법**을 배워야 해요. 그리고 지금은 그걸 도와주는 도구가 있어요."

## "AI요? 부동산에 AI를 쓴다고요?"

정대리가 의아한 표정을 지었습니다.

"ChatGPT 같은 거요? 그건 글 쓰거나 코딩할 때 쓰는 거 아니에요?"

송과장이 웃었습니다.

"저도 처음엔 그렇게 생각했어요. 근데 한번 써보니까, **부동산 분석에 진짜 강력**하더라고요."

김부장이 고개를 갸웃거렸습니다.

"AI한테 부동산 물어보면 맨날 '전문가와 상담하세요'만 하지 않아?"

송과장이 노트북을 돌려 보여주었습니다.

"**질문을 잘하면** 완전히 다른 답을 받을 수 있어요. 제가 어제 ChatGPT한테 질문한 거 보여드릴게요."

### 송과장의 질문

```
나는 연소득 1억, 현금 3억 있는 1주택자야.
2025년 10월 15일 규제 기준으로
내가 추가로 살 수 있는 아파트 최대 가격을 계산해줘.
서울 투기과열지구 기준으로.
```

### ChatGPT 답변 (요약)

```
## 1주택자 추가 매수 분석

### 규제 적용
- 서울 투기과열지구: 추가 주택 담보대출 금지
- 대출 없이 현금으로만 구매 가능

### 계산 결과
| 항목 | 금액 |
|------|------|
| 가용 현금 | 3억 원 |
| 대출 가능액 | 0원 (규제) |
| 최대 매수 가능가 | **3억 원** |

### 대안
1. 비규제지역에서 구매 시: LTV 60% 적용, 약 4.5억까지 가능
2. 기존 주택 매도 후 갈아타기: 대출 활용 가능

### 결론
서울 내 추가 매수 시 **현금 범위(3억) 내에서만** 가능합니다.
```

정대리가 감탄했습니다.

"와... 이렇게 구체적으로 답해줘요?"

송과장이 고개를 끄덕였습니다.

"**조건을 구체적으로 주면** 계산까지 해줘요. LTV, DTI, DSR 규제 다 적용해서."

김부장이 물었습니다.

"근데 이거 맞는 정보야? AI가 틀리면 어떡해?"

송과장이 진지하게 대답했습니다.

"좋은 질문이에요. **AI는 때때로 틀려요.** 그래서 검증하는 법도 같이 배워야 해요. 이 책에서 그걸 알려드릴 거예요."

## AI가 해결해주는 3가지 문제

송과장이 세 손가락을 펼쳤습니다.

"부동산 투자자들이 겪는 문제가 크게 세 가지 있어요. AI가 이걸 해결해줍니다."

### 문제 1: 정보가 여기저기 흩어져 있다

"실거래가는 호갱노노, 입주물량은 아실, 정책은 국토부... 사이트를 5개씩 돌아다녀야 해요."

정대리가 끄덕였습니다.

"맞아요. 탭만 10개 열려있어요."

송과장이 대답했습니다.

"AI한테 물어보면 **한 번에 정리**해줘요. '동탄역 롯데캐슬 84㎡ 실거래가, 전세가율, 최근 뉴스 정리해줘'라고 하면 다 모아서 표로 만들어줘요."

### 문제 2: 계산이 복잡하다

"LTV 40%, DTI 40%, 스트레스 DSR +1.5%p... 이거 다 계산하려면 엑셀 필요해요."

김부장이 끄덕였습니다.

"나는 계산 포기했어. 그냥 은행 가서 물어보지."

송과장이 웃었습니다.

"AI한테 조건 주면 **1분 안에 계산** 끝나요. '내 연봉 7천, 현금 2억으로 살 수 있는 최대 아파트 가격 계산해줘'라고 하면 돼요."

### 문제 3: 비교 분석이 힘들다

"동탄 vs 광교 vs 다산... 어디가 나한테 맞는지 비교하려면 며칠 걸려요."

정대리가 한숨을 쉬었습니다.

"저도 3개 지역 비교하려다가 포기했어요."

송과장이 대답했습니다.

"AI한테 조건 주고 '비교해줘'라고 하면 **5분 안에 비교표** 만들어줘요. 교통, 학군, 시세, 미래가치까지 다 정리해서."

김부장이 물었습니다.

"그러니까 처음부터 끝까지 AI로 투자하는 거야?"

송과장이 고개를 끄덕였습니다.

"맞아요. **'지금 집 사도 되나요?'**라는 질문에서 시작해서, **'이 아파트 5.2억에 사도 되나요?'**까지. 모든 과정을 AI와 함께 분석하는 거예요."

정대리가 조심스럽게 물었습니다.

"그러면 AI가 '사라'고 하면 사고, '사지 마라'고 하면 안 사면 되는 거예요?"

송과장의 표정이 진지해졌습니다.

"**절대 아니에요.** 이 책에서 가장 중요하게 말하고 싶은 게 그거예요."

"왜? AI가 똑똑하잖아."

송과장이 사례를 들었습니다.

"미국에 **Zillow**라는 회사가 있었어요. 미국 최대 부동산 플랫폼이에요. 이 회사가 2021년에 AI로 집을 사서 되파는 사업을 했어요."

정대리가 물었습니다.

"결과가 어땠어요?"

송과장이 대답했습니다.

"**5억 달러, 약 6,500억 원을 날렸어요.** 그리고 사업을 접었어요."

김부장이 놀랐습니다.

"AI가 그렇게 똑똑한데 왜?"

송과장이 설명했습니다.

"AI는 과거 데이터를 기반으로 예측해요. 그런데 2021년 여름, 미국 부동산 시장이 급격히 식기 시작했어요. AI는 이 변화를 감지하지 못했고, 계속 비싼 가격에 집을 샀어요. 결국 손해를 봤죠."

송과장이 웃으며 대답했습니다.

"완벽해요. AI한테 투자 결정을 맡기면 안 돼요. 하지만 AI 없이 모든 걸 직접 하면 너무 오래 걸려요. **균형**이 중요해요."

김부장이 끄덕였습니다.

"그래, 도구는 잘 쓰면 되는 거지."

# 본문 1: 부동산 기초와 가치 투자

## "부동산이 왜 이렇게 어려운 거예요?"

'6개월 안에 신혼집을 구해야 하는' 정대리(31세)가 한숨을 쉬었습니다.

"유튜브 강의도 보고, 책도 읽어봤는데... 볼수록 더 모르겠어요. 금리가 어쩌고, LTV가 어쩌고, 전세가율이 어쩌고... 용어부터 어려워요."

'서울 자가에 대기업 다녔던' 김부장(53세)이 고개를 끄덕였습니다.

"나도 30년 전에 집 샀을 때 느꼈어. 부동산은 왜 이렇게 어려운 걸까."

'부동산 투자로 월급쟁이 부자된' 송과장(38세)이 미소를 지었습니다.

"사실 부동산이 어려운 데는 **분명한 이유**가 있어요. 두 가지 특성 때문이에요."

---

## 부동산이 어려운 2가지 이유

송과장이 화이트보드에 적었습니다.

### 이유 1: 부동산은 '비교 학문'이다

"부동산은 끊임없이 **비교**를 해야 해요. 비슷한 것끼리 키 맞추기를 해야 하거든요."

정대리가 물었습니다.

"비교가 뭐가 어려워요?"

송과장이 예시를 들었습니다.

"정대리, 이 세 아파트 중에 어디가 제일 좋아 보여요?"

```
A: 마포구 한강뷰 아파트 (대단지, 역세권)
B: 성동구 한강뷰 아파트 (대단지, 학군)
C: 양천구 학군 좋은 아파트 (구축, 목동)
```

정대리가 머뭇거렸습니다.

"음... 다 좋아 보이는데요?"

송과장이 웃었습니다.

"바로 그게 문제예요. **다 좋아 보여요.** 근데 가격이 다르고, 장단점이 다르고, 미래 가치도 달라요. 이걸 비교해서 '어느 게 더 저평가됐는지' 판단해야 해요. 이 과정이 복잡하니까 어렵게 느껴지는 거예요."

### 이유 2: 부동산은 '종합 예술'이다

"부동산 가격은 **한 가지 이유로만 움직이지 않아요.** 여러 가지가 복합적으로 작용해요."

송과장이 표를 그렸습니다.

| 영향 요인 | 예시 |
|----------|------|
| **금리** | 금리 오르면 대출 부담↑, 매수 수요↓ |

# Part 0 마무리: AI 투자의 시작

## "이제 왜 AI가 필요한지 알겠어요!"

'6개월 안에 신혼집을 구해야 하는' 정대리(31세)가 노트를 펼치며 환하게 웃었습니다.

"Part 0에서 배운 거 정리해볼게요."

```
내가 배운 것:
- 부동산이 어려운 이유: 비교 학문 + 종합 예술
- 프롭테크 덕분에 데이터 접근은 쉬워졌다
- 하지만 데이터 '분석'은 여전히 어렵다
- AI가 이 gap을 채워준다
- 단, AI를 맹신하면 안 된다 (Zillow 사례)
```

'서울 자가에 대기업 다녔던' 김부장(53세)도 고개를 끄덕였습니다.

"나도 정리됐어. 프롭테크로 정보는 볼 수 있는데, 그걸 어떻게 해석하고 판단해야 하는지가 문제였거든. AI가 그걸 도와주는 거구나."

'부동산 투자로 월급쟁이 부자된' 송과장(38세)이 웃으며 말했습니다.

"두 분 다 핵심을 잘 잡으셨네요. Part 0의 목표는 **'왜 AI인가'**를 이해하는 거였어요."

## Part 0 핵심 정리: 5가지 깨달음

송과장이 화이트보드에 최종 정리를 했습니다.

### 1. 데이터는 공개됐지만, 해석 능력이 격차를 만든다

"실거래가, 전세가율, 입주물량... 다 공개되어 있어요. 하지만 **같은 데이터를 보고도** 어떤 사람은 8억 수익, 어떤 사람은 5천만 원 손실을 봐요."

정대리가 물었습니다.

"그 차이가 뭐예요?"

"**핵심 변수를 파악하는 능력**이에요. GTX가 지나간다는 정보만 본 사람과, '강남 접근성이 얼마나 개선되나'까지 본 사람의 차이죠."

### 2. 프롭테크는 정보를 보여주고, AI는 분석을 해준다

"호갱노노, 아실, 네이버 부동산... 훌륭한 도구예요. 하지만 **'내 조건에 맞는 분석'**은 안 해줘요. AI는 그걸 해줘요."

| 도구 | 역할 |
|------|------|
| 프롭테크 | 데이터 **보여주기** |
| AI | 데이터 **분석하기** |

### 3. AI로 시간, 비용, 에너지, 정보 4가지를 아낄 수 있다

| 이득 | 효과 |
|------|------|
| 시간 | 임장 시간 60% 절감 |
| 비용 | 연 70만 원+ 절감 |
| 에너지 | 꾸준한 투자 공부 가능 |
| 정보 | 사이트 5개 → 질문 1개 |

### 4. 하지만 AI를 맹신하면 안 된다

"Zillow는 AI로 집을 사고팔다가 **6,500억 원을 날렸어요.** AI는 과거 데이터에 의존하기 때문에 급변 상황에 약해요."

김부장이 끄덕였습니다.

"결국 **최종 판단은 내가** 해야 하는 거구나."

### 5. AI는 도구, 책임은 투자자

송과장이 강조했습니다.

"이 책 전체에서 반복할 원칙이에요."

```
✅ AI는 정보 정리와 계산을 빠르게 해주는 '도구'
✅ 빠른 정리는 AI, 최종 확인은 공식 사이트
✅ 투자 결정의 책임은 본인
```
