# Part 1: ChatGPT가 당신의 투자 비서가 된다

## "AI한테 부동산 물어봤는데, 너무 뜬구름 잡는 답만 해요"

'6개월 안에 신혼집을 구해야 하는' 정대리(31세)가 스마트폰을 내려놓으며 한숨을 쉬었습니다.

지난주, 그는 ChatGPT에게 부동산 질문을 던져봤습니다.

```
나: "지금 집 사도 되나요?"

ChatGPT: "부동산 구매 결정은 개인의 재정 상황, 시장 상황,
장기적인 계획 등 여러 요소를 고려해야 합니다.
전문가와 상담하시는 것을 권장합니다."
```

정대리가 답답한 표정으로 말했습니다.

"이런 답이면 물어볼 필요가 없잖아요. 전문가 상담하라는 건 나도 아는데..."

'서울 자가에 대기업 다녔던' 김부장(53세)이 고개를 끄덕였습니다.

"나도 처음에 그랬어. ChatGPT한테 '노원구 재건축 어때?'라고 물어봤는데, 교과서 같은 답만 주더라고."

'부동산 투자로 월급쟁이 부자된' 송과장(38세)이 웃으며 끼어들었습니다.

"두 분 다 **질문을 잘못한 거예요.**"

정대리와 김부장이 동시에 눈을 크게 떴습니다.

"질문이요?"

## "AI는 질문하는 만큼만 답해요"

송과장이 노트북을 열었습니다.

"AI는 **만능이 아니에요.** 하지만 **제대로 질문하면** 전문가 수준의 답을 받을 수 있어요."

화이트보드에 두 가지 질문을 적었습니다.

```
❌ 나쁜 질문:
"지금 집 사도 되나요?"
→ 너무 모호해서 AI가 일반론으로 대답

✅ 좋은 질문:
"나는 연소득 7천만원, 현금 3억 있는 무주택자야.
2025년 10월 대출 규제 기준으로
내가 살 수 있는 최대 아파트 가격을 계산해줘.
LTV 40%, 스트레스 DSR 적용해서."
→ 구체적인 조건이 있어서 AI가 정확한 계산 가능
```

정대리의 눈이 반짝였습니다.

"아, 조건을 자세히 말해야 하는 거구나!"

김부장도 끄덕였습니다.

"그러고 보니 부하직원한테 업무 지시할 때도 자세히 말해야 제대로 하더라고. AI도 마찬가지인 거구나."

송과장이 고개를 끄덕였습니다.

"**맞아요. AI는 똑똑한 신입사원**과 같아요. 능력은 있는데, 무엇을 해야 하는지 **명확하게 알려줘야** 일을 제대로 해요."

## ChatGPT가 해줄 수 있는 5가지

정대리가 펜을 들었습니다.

"구체적으로 ChatGPT가 부동산 투자에서 뭘 해줄 수 있어요?"

송과장이 다섯 손가락을 펼쳤습니다.

### 1. 내 구매력 자동 계산

"**'내가 얼마짜리 집을 살 수 있을까?'** 이 질문에 1분 안에 답해줘요."

| 필요 정보 | 예시 |
|----------|------|
| 가용현금 | 3억 원 |
| 연소득 | 7천만 원 |
| 기존 대출 | 없음 |
| 보유 주택 | 0채 |

→ LTV, DTI, 스트레스 DSR 규제를 모두 적용해서 **최대 구매 가능 금액**을 계산해줘요.

### 2. 수많은 지역 비교 분석

"**동탄, 광교, 다산, 의정부... 어디가 좋을까?** 이런 질문에 5분 만에 비교표를 만들어줘요."

| 비교 항목 | 내용 |
|----------|------|
| 교통 | GTX, 지하철 접근성 |
| 학군 | 학원가, 학교 순위 |
| 시세 | 현재 가격, 상승률 |
| 미래가치 | 개발 계획, 인프라 |

### 3. 적정가 산출

"**이 아파트 13억이 비싼 건가, 싼 건가?** 실거래가를 분석해서 적정 가격 범위를 알려줘요."

### 4. AI 원격 임장

"**주말마다 차 몰고 돌아다닐 필요 없어요.** AI가 후보 단지의 교통, 인프라, 주민 평가까지 분석해줘요."

### 5. 최종 의사결정 지원

"**'그래서 이 집 사야 해, 말아야 해?'** 모든 분석을 종합해서 Buy/Pass/Negotiate 판단을 도와줘요."

정대리의 눈이 반짝였습니다.

"이걸 다 AI가 해준다고요?"

송과장이 웃으며 대답했습니다.

"**제대로 질문하면** 다 해줘요. 이 책을 끝까지 따라오면 이 5가지 능력을 모두 갖출 수 있어요."

# 섹션 ① AI 도구 비교: 어떤 AI를 써야 할까?

## "ChatGPT만 쓰면 안 되나요?"

'6개월 안에 신혼집을 구해야 하는' 정대리(31세)가 질문했습니다.

"요즘 AI가 많잖아요. ChatGPT, Claude, Gemini, Perplexity... 다 비슷한 거 아니에요? 그냥 ChatGPT 하나만 쓰면 안 되나요?"

'서울 자가에 대기업 다녔던' 김부장(53세)도 궁금한 표정을 지었습니다.

"나도 그게 궁금해. 회사에서 ChatGPT 쓰라고 해서 쓰긴 하는데, 다른 것들은 뭐가 다른 건지 모르겠더라고."

'부동산 투자로 월급쟁이 부자된' 송과장(38세)이 노트북을 열며 대답했습니다.

"**다 비슷해 보이지만, 잘하는 게 달라요.** 마치 식당마다 대표 메뉴가 다른 것처럼요. 비빔밥 잘하는 집이 있고, 삼겹살 잘하는 집이 있잖아요?"

정대리가 고개를 끄덕였습니다.

"그러니까... AI마다 잘하는 분야가 다르다는 거예요?"

송과장이 미소를 지었습니다.

"정확해요. 그래서 **부동산 투자에 어떤 AI가 어울리는지** 비교해볼게요."

## 5대 AI 도구 한눈에 비교

송과장이 화이트보드에 표를 그렸습니다.

"2025년 12월 기준, 부동산 투자에 쓸 만한 AI 5가지예요."

| AI 도구 | 개발사 | 핵심 강점 | 부동산 투자 활용 |
|---------|--------|----------|-----------------|
| **ChatGPT** | OpenAI | 범용 + 계산 + 한국어 | 대출 계산, 수익률 시뮬레이션 |
| **Claude** | Anthropic | 긴 문서 + 논리적 분석 | 지역 비교, 보고서 작성 |

"다섯 개나 돼? 다 써야 해?"

## AI별 '이것만은 최고'

송과장이 각 AI의 강점을 설명했습니다.

### 1. ChatGPT: "계산이 필요하면 얘 불러"

"ChatGPT의 핵심 무기는 **Code Interpreter**예요. 복잡한 계산을 코드로 돌려줘요."

| 활용 예시 | 설명 |
|----------|------|
| 대출 한도 계산 | LTV, DTI, DSR 규제 적용해서 정확히 계산 |
| 수익률 시뮬레이션 | 3년 보유, 5년 보유 시 시나리오별 수익률 |
| 세금 계산 | 취득세, 양도세 시뮬레이션 |
| 차트 생성 | 시세 추이를 그래프로 시각화 |

정대리가 눈을 반짝였습니다.

"와, 그러니까 엑셀 없이도 계산할 수 있는 거예요?"

"맞아요. ChatGPT한테 '내 조건으로 대출 한도 계산해줘'라고 하면, 코드를 돌려서 정확한 숫자를 알려줘요."

### 2. Claude: "길게 분석하려면 얘 불러"

"Claude는 **200K 토큰**까지 한 번에 읽어요. ChatGPT의 128K보다 훨씬 길어요."

김부장이 질문했습니다.

"토큰이 뭐야?"

"쉽게 말하면, **한 번에 읽고 기억할 수 있는 글자 수**예요. Claude는 A4 기준 약 300페이지를 한 번에 분석할 수 있어요."

| 활용 예시 | 설명 |
|----------|------|
| 지역 비교 분석 | 5개 지역 데이터를 한 번에 비교 |
| 긴 보고서 작성 | 투자 분석 보고서 초안 작성 |
| PDF 계약서 분석 | 아파트 분양 계약서 검토 |
| 논리적 구조화 | 복잡한 정보를 표로 정리 |

송과장이 덧붙였습니다.

"특히 **'이 지역 vs 저 지역 비교해줘'** 같은 질문에 Claude가 강해요. 여러 항목을 체계적으로 정리해주거든요."

### 3. Gemini: "구글 검색이랑 연동됨"

"Gemini는 **Google과 연동**돼요. 검색하면서 답변해줘요."

| 활용 예시 | 설명 |
|----------|------|
| 정책 검색 | "2025년 10월 부동산 대책 알려줘" |
| 이미지 분석 | 평면도, 단지 배치도 분석 |
| 유튜브 요약 | 부동산 유튜브 영상 내용 정리 |

정대리가 물었습니다.

"그러면 Gemini가 검색은 제일 잘하는 거예요?"

송과장이 고개를 저었습니다.

"검색 **'만'** 보면 Perplexity가 더 강해요. Gemini는 검색 + 분석을 **같이** 할 때 편해요."

### 4. Perplexity: "검색의 신"

"Perplexity는 **실시간 검색 전문**이에요. 모든 답변에 출처 링크가 붙어요."

| 활용 예시 | 설명 |
|----------|------|
| 최신 뉴스 | "이번 주 부동산 뉴스 정리해줘" |
| 시세 조회 | "강남 래미안 현재 시세" |
| 정책 확인 | "스트레스 DSR 적용 기준" |
| 출처 검증 | 모든 답변에 원문 링크 제공 |

## Part 1에서 배운 것: 핵심 정리

송과장이 마지막으로 정리했습니다.

### 1. AI는 질문하는 만큼만 답한다

"**좋은 질문 = 좋은 답변**이에요. '집 사도 돼?'보다 '내 조건으로 구매 가능가 계산해줘'가 훨씬 유용한 답변을 얻어요."

### 2. AI마다 잘하는 게 다르다

| AI | 강점 |
|-----|------|
| ChatGPT | 계산, 분석, 범용 |
| Claude | 긴 문서, 비교 분석 |
| Perplexity | 최신 정보, 출처 |
| Gemini | Google 연동 |
| Comet | 멀티 AI 비교 |

### 3. Custom Instructions = 한 번 설정, 평생 편리

"**투자 조건을 미리 입력해두면**, 매번 설명할 필요 없이 맞춤 분석을 받을 수 있어요."

### 4. ChatGPT Plus로 시작하면 충분

"여러 AI를 다 쓸 필요 없어요. **ChatGPT Plus 하나로 80%는 해결**됩니다. 나머지는 필요할 때 추가하면 돼요."
